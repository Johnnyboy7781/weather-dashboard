# Welcome to the Weather Dashboard ☁

## Purpose
A responsive website that pulls information from OpenWeather's Weather and Geocoding APIs. View the OpenWeather API docs here: https://openweathermap.org/api.

* HTML
* CSS
* JavaScript
* LocalStorage
